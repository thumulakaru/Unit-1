# Conditions if, then, else
## Minimum of two numbers
```.py
a = int(input())
b = int(input())

if a > b:
    print(b)
else:
    print(a)
```

## Sign function
```.py
a = int(input())
if a>0:
    print("+1")
elif a==0:
    print(0)
else:
    print("-1")
```
