# School desks
## Solution

```.py
a = int(input("Enter class a students:"))
b = int(input("Enter class b students:"))
c = int(input("Enter class c students:"))

da = (a//2 + a % 2)
db = (b//2 + b % 2)
dc = (c//2 + c % 2)

print(da+db+dc)
```
