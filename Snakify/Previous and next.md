# Previous and next
## Solution to the problem
```.py
a = int(input("Please enter the number:"))
b = (str(a+1))
c = (str(a-1))
d = str(a)
print("The next number for the number " + d + " is " + b +".")
print("The previous number for the number " + d + " is " + c+".")
```
